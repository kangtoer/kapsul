# 🏺 KAPSUL WAKTU NUSANTARA
## Sistem Eksplorasi Arkeologi Fiksi v3.1

---

## 📌 Ringkasan Proyek

**KAPSUL WAKTU NUSANTARA** adalah platform pembelajaran interaktif yang menggabungkan **gamifikasi** dengan eksplorasi arkeologi fiksi untuk meningkatkan pemahaman siswa tentang sejarah, budaya, dan warisan Nusantara. Platform ini dirancang khusus untuk **pendidik profesional** yang ingin menciptakan pengalaman belajar yang menarik, interaktif, dan bermakna dalam konteks **social science education**.

---

## ✨ Fitur Utama

### 1. 🎮 **Sistem Gamifikasi Pembelajaran**
- **Poin Pengalaman (XP)**: Siswa mendapatkan poin untuk setiap pencapaian
- **Level & Progression**: Sistem level bertingkat dari Peneliti Pemula hingga Ahli Arkeologi
- **Achievement Badges**: Lencana prestasi untuk milestone khusus
- **Leaderboard**: Papan peringkat kelas untuk motivasi sehat antar siswa
- **Daily Quests**: Misi harian yang mendorong engagement berkelanjutan

### 2. 📚 **Modul Pembelajaran Interaktif**
- **Timeline Interaktif Nusantara**: Jelajahi periode sejarah dari era prasejarah hingga modern
- **Artefak Digital**: Koleksi 3D objek arkeologi dari berbagai museum virtual
- **Cerita Fiksi Berbasis Fakta**: Narasi edukatif yang menghubungkan sejarah dengan kehidupan kontemporer
- **Kuis Adaptif**: Pertanyaan yang menyesuaikan dengan level pemahaman siswa

### 3. 🎯 **Eksplorasi Arkeologi Fiksi**
- **Mini Excavation Games**: Simulasi penggalian arkeologi interaktif
- **Mystery Solving**: Pemecahan misteri sejarah melalui clue-gathering
- **Time Capsule Opening**: Membuka "kapsul waktu" dengan penemuan bersejarah
- **Archaeological Reports**: Membuat laporan penelitian hasil eksplorasi

### 4. 👥 **Kolaborasi & Komunitas**
- **Group Expeditions**: Misi kelompok untuk pembelajaran kolaboratif
- **Discussion Forums**: Forum diskusi per topik pembelajaran
- **Peer Reviews**: Sistem review antar siswa untuk hasil penelitian
- **Teacher Dashboard**: Panel kontrol lengkap untuk pendidik

### 5. 📊 **Analytics & Progress Tracking**
- **Real-time Performance**: Monitor kemajuan siswa secara real-time
- **Learning Analytics**: Grafik detail tentang pola belajar siswa
- **Content Mastery Reports**: Laporan penguasaan materi per topik
- **Customizable Reports**: Export laporan dalam berbagai format

## 🚀 Cara Penggunaan

### A. Untuk Pendidik (Teacher Setup)

#### Langkah 1: Registrasi & Setup Kelas
```
1. Daftar sebagai pendidik di platform
2. Buat kelas baru dengan nama dan deskripsi
3. Tentukan level siswa (SD/SMP/SMA/Universitas)
4. Atur periode pembelajaran (semester/tahun akademik)
5. Sesuaikan sistem poin (XP multiplier, bonus quest)
```

#### Langkah 2: Kurasi Konten Pembelajaran
```
1. Pilih modul dari library yang tersedia
2. Sesuaikan dengan kurikulum lokal Anda
3. Tambahkan materi khusus dari arsip lokal
4. Set difficulty level untuk setiap modul
5. Tentukan deadline dan milestone pembelajaran
```

#### Langkah 3: Konfigurasi Gamifikasi
```
1. Buka menu "Gamification Settings"
2. Tentukan sistem reward:
   - Base XP per aktivitas (5-50 poin)
   - Bonus untuk kolaborasi (10-100 poin)
   - Streak bonus (berturut-turut menyelesaikan)
3. Atur achievement badges yang relevan
4. Aktifkan/nonaktifkan leaderboard
5. Set weekly challenges & special events
```

#### Langkah 4: Monitor Pembelajaran
```
1. Kunjungi Teacher Dashboard setiap hari
2. Lihat aktivitas siswa real-time
3. Identifikasi siswa yang tertinggal
4. Berikan feedback & encouragement
5. Generate report bulanan untuk kepala sekolah
```

### B. Untuk Siswa (Student Learning Journey)

#### Fase 1: Onboarding & Orientation
```
1. Buat profil sebagai "Peneliti Arkeologi"
2. Pilih avatar & tim eksplorasi
3. Ikuti tutorial mini (5-10 menit)
4. Selesaikan "First Excavation" untuk warm-up
5. Lihat papan misi pertama Anda
```

#### Fase 2: Memulai Eksplorasi
```
1. Buka Timeline Nusantara
2. Pilih periode sejarah yang menarik
3. Baca narasi fiksi yang disediakan
4. Eksplorasi artefak digital 3D
5. Jawab pertanyaan pemahaman (dapatkan XP)
```

#### Fase 3: Mini-Games & Excavation
```
1. Mainkan "Dig Site Simulator" untuk periode pilihan
2. Selesaikan puzzle arkeologi (matching, logic)
3. Kumpulkan clue untuk mystery story
4. Buat sketsa & catatan di "Field Journal"
5. Submit untuk mendapat XP & achievement
```

#### Fase 4: Kolaborasi & Kompetisi
```
1. Bergabung dengan grup eksplorasi kelas
2. Selesaikan "Group Quest" bersama teman
3. Lihat posisi di leaderboard
4. Hadir di "Weekly Challenge" untuk bonus XP
5. Bantu teman melalui forum diskusi
```

#### Fase 5: Laporan & Presentasi
```
1. Tulis laporan arkeologi di modul Report Builder
2. Sertakan temuan, analisis, dan kesimpulan
3. Minta peer review dari 2 teman sekelas
4. Revisi berdasarkan feedback
5. Submit final report untuk assessment
```

## 🎮 Sistem Gamifikasi Pembelajaran Lengkap

### A. **Struktur Sistem Poin**

| Aktivitas | Base XP | Bonus | Total Max |
|-----------|---------|-------|-----------|
| Menyelesaikan Modul | 50 | +25 (kolaborasi) | 75 |
| Menjawab Kuis | 10 | +5 (perfect score) | 15 |
| Mini-Game Excavation | 30 | +15 (first time) | 45 |
| Group Quest | 40 | +20 (semua anggota selesai) | 60 |
| Forum Discussion | 5 | +10 (helpful answer) | 15 |
| Peer Review | 8 | +7 (quality review) | 15 |
| Weekly Challenge | 100 | +50 (top 3) | 150 |

### B. **Level Progression System**

```
Level 1: Peneliti Pemula (0-500 XP)
Level 2: Penjelajah Junior (501-1,500 XP)
Level 3: Asisten Arkeolog (1,501-3,500 XP)
Level 4: Ahli Arkeologi Dasar (3,501-7,000 XP)
Level 5: Ahli Arkeologi Senior (7,001-12,000 XP)
Level 6: Profesor Arkeologi Nusantara (12,001+ XP)
```

### C. **Achievement Badges**

🏅 **Bronze Tier:**
- "First Dig" - Selesaikan excavation pertama
- "Knowledge Seeker" - Kumpulkan 100 XP
- "Team Player" - Selesaikan 1 group quest

🥈 **Silver Tier:**
- "Master Explorer" - Kumpulkan 2,000 XP
- "Historian in Training" - Selesaikan 5 modul
- "Leaderboard Star" - Ranking top 10 selama 2 minggu

🥇 **Gold Tier:**
- "Archaeological Expert" - Kumpulkan 10,000 XP
- "Nusantara Scholar" - Selesaikan semua modul
- "Community Champion" - Berikan 50 peer reviews berkualitas

### D. **Challenge & Event System**

**Daily Quests** (Misi Harian):
- Completion: 20 XP
- Streak Bonus: +10 XP per hari berturut-turut (max 7 hari = +70 XP)

**Weekly Challenge** (Tantangan Mingguan):
- Theme: Berbeda setiap minggu (misal: "Era Majapahit Week")
- Hadiah: 100-150 XP + Special Badge
- Format: Kombinasi kuis, mini-game, dan creative task

**Monthly Tournament** (Turnamen Bulanan):
- Team vs Team competition
- Multiple events (speed challenges, creativity contests)
- Grand Prize: Special title + 500 XP + exclusive badge

**Seasonal Events** (Event Musiman):
- Perayaan hari jadi lokal
- Theme-based learning modules
- Limited-time exclusive rewards

## 💡 Mengapa Anda Harus Mencobanya?

### 🎯 **Untuk Pendidik**

#### 1. **Meningkatkan Engagement Siswa Drastis**
- Gamifikasi terbukti meningkatkan partisipasi hingga **87%** (Gartner, 2023)
- Siswa yang tadinya pasif menjadi aktif berkompetisi & kolaborasi
- Attendance & assignment completion meningkat signifikan

#### 2. **Hemat Waktu Persiapan & Grading**
- Automated quiz grading menghemat waktu hingga **10 jam/minggu**
- Pre-built modules langsung dapat digunakan
- Analytics dashboard menunjukkan progress tanpa perlu menghitung manual

#### 3. **Sesuai dengan Kurikulum Modern**
- Aligned dengan **Profil Pelajar Pancasila** (Indonesia)
- Mengembangkan keterampilan abad ke-21: kolaborasi, critical thinking, creativity
- Mendukung social science education yang holistik

#### 4. **Data-Driven Teaching Insights**
- Identifikasi siswa yang struggle lebih awal
- Personalisasi pembelajaran berdasarkan learning patterns
- Evidence-based decisions untuk intervention strategies

#### 5. **Meningkatkan Hasil Assessment**
- Siswa lebih tersiap untuk tes karena belajar sambil bermain
- Retainsi pengetahuan meningkat (dari spaced repetition dalam games)
- Motivasi intrinsik meningkat → pembelajaran lebih dalam

## 🔧 Instalasi & Setup

### Quick Start untuk Pendidik

```bash
# 1. Clone repository
git clone https://github.com/kangtoer/kapsul.git
cd kapsul

# 2. Install dependencies
npm install

# 3. Konfigurasi environment
cp .env.example .env
# Edit .env dengan database credentials Anda

# 4. Setup database
npm run migrate
npm run seed

# 5. Start server
npm run dev

# 6. Akses di http://localhost:3000
# Login dengan akun demo teacher
# Email: teacher@example.com
# Password: demo123
```

### Untuk Siswa
- Cukup akses link yang dibagikan guru
- Tidak perlu install apapun
- Bisa dari browser atau mobile app
